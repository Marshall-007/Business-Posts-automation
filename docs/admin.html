Your companies

Every business you automate, with one-click access to its dashboard and setup pages.

How this works
  • Each company lives in its own GitHub repository (a copy of this automation), with its own content folders, schedule, captions, and platform secrets.
  • Open dashboard switches this browser to that company and opens its posting dashboard.
  • The setup checklist on each card tracks what is still missing (secrets, Pages hosting, platform connections).

Set up a new company

Creates a ready-to-run copy of this whole automation for a new client: repo, workflows, dashboard, captions - personalized with their business details.

How this works, step by step
  • Marks this repository as a GitHub template, then creates a fresh copy under your account for the new client (public, so Instagram can fetch images).
  • Writes their business profile into config.yaml: name, description, phone and website (used in every caption), plus starter topics and hashtags.
  • Clears the starter content and queue so the client begins empty.
  • Then a checklist appears: add their platform secrets, switch on Pages hosting, and connect Instagram/Facebook/TikTok exactly like the first setup.
  • The token needs permission to create repositories (fine-grained PAT with Administration and Contents read/write, or a classic token with the repo scope).
